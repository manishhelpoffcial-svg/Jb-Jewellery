## 🚫 SUSPEND / ACTIVATE ACCOUNT

```
SUSPEND ACCOUNT:
- Admin clicks Suspend button
- Confirmation popup:
  "Are you sure you want to
   suspend Priya Sharma's account?
   They will not be able to login."
- [Cancel] [Yes, Suspend]
- On confirm:
  Set isActive: false in Replit DB
  User sees "Account suspended"
  on login attempt
  Email sent to customer:
  "Your account has been suspended.
   Contact us for help."

ACTIVATE ACCOUNT:
- Admin clicks Activate
- Confirmation popup
- Set isActive: true in Replit DB
- Email sent to customer:
  "Your account has been reactivated!
   You can login now."
```
