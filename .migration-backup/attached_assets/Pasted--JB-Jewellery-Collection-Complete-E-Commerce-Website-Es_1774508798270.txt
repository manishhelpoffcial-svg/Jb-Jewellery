
## JB Jewellery Collection
### Complete E-Commerce Website — Estallo Style
### White & Yellow Theme | WhatsApp Checkout | Firebase Backend | Admin Panel

---

## 🎯 PROJECT OVERVIEW

Build a complete professional e-commerce website exactly like **estailofashion.com** in layout and structure. Brand name is **JB Jewellery Collection** (short: **JB**). Niche is artificial/fashion jewellery. Color theme is **White & Yellow only**. Checkout is via **WhatsApp**. Backend is **Firebase**. Admin panel is fully included.

### Admin Orders `/admin/orders`:
```
Table columns:
Order ID | Date | Customer | Phone |
Items Count | Total | Status | Actions

Filter: All | Pending | Confirmed |
        Processing | Shipped | Delivered | Cancelled

Each row Actions:
- [View] → opens order detail modal
- [Update Status] dropdown
- [WhatsApp Customer] quick link

Order Detail Modal:
- Full order info
- All items with images
- Price breakdown
- Delivery address
- Status update dropdown
- [Send Status Update] button
- [Download Invoice] button
- [WhatsApp Customer] button
```

### Admin Products `/admin/products`:
```
Table: Image | Name | Category |
       MRP | Final Price | Stock |
       Status | Actions

Actions per product:
- ✏️ Edit
- 🗑️ Delete (with confirmation)
- 👁️ Toggle active/inactive

Add/Edit Product Form:
- Product Name *
- Category dropdown *
- MRP Price *
- Discount % → auto calculates Final Price
- Final Price (auto-calculated, editable)
- Description *
- Material / Care Instructions
- Stock Quantity *
- Upload Images (multiple, to Firebase Storage)
- Tags: [New] [Bestseller] [Featured] [Sale]
- Status: Active / Inactive
- [Save Product] yellow button
```

### Admin Customers `/admin/customers`:
```
Table: Name | Email | Phone |
       Orders | Total Spent | Joined | Actions

Actions:
- View Profile
- View Order History
- WhatsApp Customer
```

### Admin Coupons `/admin/coupons`:
```
Table: Code | Type | Value |
       Min Order | Expiry | Status | Actions

Create Coupon Form:
- Code (e.g. JBFIRST)
- Type: Flat ₹ / Percentage %
- Discount Value
- Minimum Order Amount
- Expiry Date
- [Create] yellow button
```

### Admin WhatsApp Order Response:
```
When admin receives WhatsApp order:

If admin replies: ACCEPT JB2025001
→ System updates order status to "Confirmed"
→ Customer receives WhatsApp message:
  "✅ Your JB Jewellery order #JB2025001
   has been CONFIRMED! 🎉
   We'll ship it soon. Track at:
   jbjewellery.com/order/JB2025001"

If admin replies: DECLINE JB2025001
→ System updates order status to "Cancelled"
→ Customer receives WhatsApp message:
  "❌ Sorry, your order #JB2025001
   could not be processed.
   Contact us: wa.me/91XXXXXXXXXX"
```
